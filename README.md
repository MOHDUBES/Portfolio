# Portfolio | MOHD UBES

A modern, responsive, and visually striking personal portfolio website built for **Mohd Ubes**, a Full Stack Web Developer & IT student at MAIT.

## 🚀 Features
- **Modern UI/UX**: Sleek dark-mode aesthetic with cyan and purple gradients.
- **Glassmorphism**: Beautiful frosted glass effects on the navigation and interactive cards.
- **Fully Responsive**: Optimized for desktops, tablets, and mobile devices with a seamless custom mobile navigation overlay.
- **Dynamic Animations**: Smooth scroll fade-ins, hover effects, and modern transitions.
- **Live Resume Integration**: Options to instantly download the resume as a PDF or view it online directly from the hero section.

## 💻 Technologies Used
- **HTML5**: Semantic markup for better accessibility and SEO.
- **CSS3**: Custom variables, Flexbox, CSS Grid, responsive media queries, and modern styling techniques (built completely without external frameworks).
- **Vanilla JavaScript**: For scroll animations (Intersection Observer), active link highlighting, and mobile menu toggling.

## 🛠️ How to Run Locally

1. Clone this repository or download the folder.
2. Open the folder in your terminal.
3. Make sure you have Node.js installed.
4. Start the local development server with live-reloading:
   ```bash
   npm start
   ```
   *(This uses browser-sync to serve your files and gives you a local URL and a network URL to test on your phone).*
